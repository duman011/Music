This application is a beta version that enables music streaming using the Deezer API.

## Using Technologies:
- UIKit
- MVVM-C
- Deezer API
- Alamofire
- Kingfisher
- FirebaseAuth
- FirebaseFirestore
- FirebaseStorage
- Lottie

## 3rd Party Libraries

This app is built with the help of the following 3rd party libraries:

- Alamofire (5.8.1)
- Firebase (10.17.0)
- Kingfisher (7.10.0)
- Lottie (4.3.3)
